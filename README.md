# DISTAK ERP 2.0

Site/app estática para publicar no GitHub Pages ou no Render.

## Ficheiros principais
- `index.html` — página principal da aplicação.
- `assets/styles.css` — desenho, cores e responsivo.
- `assets/app.js` — navegação, login visual e tabelas.
- `data/sample-data.js` — dados iniciais editáveis.
- `docs/relatorio-tecnico-final-modelo.md` — modelo de relatório técnico final.

## Como atualizar no GitHub
1. Entrar no repositório do site.
2. Apagar/substituir os ficheiros antigos.
3. Enviar todos os ficheiros deste pacote mantendo as pastas.
4. Confirmar que o ficheiro principal se chama `index.html`.
5. No GitHub Pages, escolher Branch `main` e pasta `/root`.
